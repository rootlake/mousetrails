# Mouse Trails

An interactive web app showcasing various mouse cursor effects and trails.

## Features

- **Particles**: Floating particles that follow your mouse with physics
- **Rainbow Trail**: Colorful trail that follows your cursor
- **Sparkles**: Twinkling sparkle effects
- **Glow**: Soft glow effect at cursor position
- **Ripple**: Expanding ripple effects
- **Random Background**: Click to change the background color

## Usage

Simply open `index.html` in a web browser and move your mouse around. Toggle different effects on and off using the buttons.

## Hosting on GitHub Pages

1. Push this repository to GitHub
2. Go to Settings > Pages in your repository
3. Select the branch and folder (usually `main` branch, `/root`)
4. Your site will be available at `https://yourusername.github.io/MouseTrails/`
